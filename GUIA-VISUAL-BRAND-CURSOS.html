O que é HTML

HTML significa HyperText Markup Language. É a linguagem padrão para criar páginas na web.

  • Estrutura o conteúdo em elementos semânticos
  • Funciona junto com CSS (estilo) e JavaScript (interatividade)
  • É interpretado pelos navegadores
slide_exemplo_pratico
Exemplo: Primeira Página

Aqui está uma página HTML completa:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
<title>Minha Primeira Página</title>
</head>
<body>
<h1>Olá, mundo!</h1>
</body>
</html>

Crie sua primeira página

Reproduza o exemplo acima. Depois, altere o título e a mensagem de boas-vindas com seu nome.

Dica: Abra um editor de texto (VS Code, Notepad++) e salve com extensão .html. Depois abra no navegador com Ctrl+O.

Por que HTML é o primeiro?

HTML é o fundamento da web. Todo desenvolvedor web precisa dominar. CSS vem depois (estilo). JavaScript vem depois (interatividade). Juntos formam a tríade do front-end.

Checklist — Cada Aula

Criação

  • Seguir template de estrutura (7 componentes acima)
  • Usar Geom para títulos e corpo
  • Usar JetBrains Mono SÓ para código/labels técnicos
  • Fundo BRANCO #FBFCFE
  • Textos em AZUL ELÉTRICO #2D5BFF (títulos) ou TINTA #0F1419 (corpo)
  • Destaques em CIANO #0BC5C5
  • Incluir objetivo de aprendizagem explícito
  • Incluir pelo menos 1 exemplo prático com código/mockup
  • Incluir 1 exercício/desafio com enunciado claro
  • Incluir resumo final antes do rodapé

Revisão (antes de publicar)

  • Linguagem humana (sem IA detectável)
  • Sem travessão '—' ou ' - ' como pausa
  • Tecnologias atualizadas (pesquisar versão atual do ano)
  • Desafios calibrados (fixação → aplicação → análise)
  • Alt text em TODAS as imagens
  • Glossário de termos técnicos novos
  • Nível de leitura apropriado (ensino médio, 14-17 anos)

Publicação

  • Campo 'revisado' = true
  • Imagem de capa (1920x1080px)
  • Descrição breve do curso (<150 caracteres)
  • Metadados: total de aulas, duração estimada, nível
  • Rodapé visível: 'Professor André Gomes • Desenvolvedor e Educador'
  • Logo { } AG visível na capa

Observações Importantes

Regras de Ouro (Não quebrar!)
  • Fundo BRANCO (#FBFCFE) em TODAS as aulas
  • Nunca usar cinza como fundo — ilegível e quebra a marca
  • Geom é font primária — títulos E corpo. JetBrains Mono SÓ código
  • Logo { } AG visível em toda capa de curso
  • Rodapé padrão: "Professor André Gomes • Desenvolvedor e Educador"
  • Contraste mínimo 4.5:1 (WCAG AA) — azul sobre branco passa ✓

⚠️ Sobre Alt Text: Toda imagem DEVE ter descrição acessível. "imagem1.png" não é alt text. Escreva o que a imagem mostra (ex: "Exemplo de página HTML com título e parágrafo").
