## 🎯 Identidade Visual

### Paleta de Cores
```
Azul Elétrico  #2D5BFF   ← Primária (títulos, destaque)
Azul Claro     #4F77FF   ← Elemento visual
Tinta          #0F1419   ← Texto corpo (dark theme)
Ciano          #0BC5C5   ← Destaques, IA, ao vivo
Papel          #FBFCFE   ← Fundo branco (superfície)
Linha          #E6EAF0   ← Bordas e divisores
Texto Suave    #5A6675   ← Textos secundários
```

### Tipografia
- **Títulos/Corpo**: Geom (fonte da marca)
- **Labels Técnicas**: JetBrains Mono (via Google Fonts)

### Símbolo
- Chaves `{ }` abraçando `AG` = Código + Acolhimento

## 📐 Regras de Uso

### ✅ Faça assim
- Use **azul elétrico (#2D5BFF)** para todas as chamadas à ação
- **Fundo branco (#FBFCFE)** em superfícies principais
- **Ciano (#0BC5C5)** para destaques e feedback positivo
- **Texto suave (#5A6675)** para informações secundárias
- **Geom** para todos os textos de corpo e títulos
- **JetBrains Mono** apenas para código e labels técnicas

### ❌ Não faça assim
- ❌ Usar cores desaturadas ou cinza escuro para texto principal
- ❌ Misturar com cores da escola anterior (azul #1a3a5c)
- ❌ Fundos cinza ou cinza pálido
- ❌ Texto em branco sobre fundo escuro (usar na capa apenas)
- ❌ Distorcer a proporção do símbolo AG { }
